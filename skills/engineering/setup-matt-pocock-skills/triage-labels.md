# Triage Labels

skill 使用五个标准 triage 角色。本文件把角色映射到本仓库 issue tracker 的真实标签。

| mattpocock/skills 角色 | tracker 标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要 maintainer 评估 |
| `needs-info` | `needs-info` | 等待报告人补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交给 AFK agent |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个角色时，使用本表右列标签。

如果你的 tracker 使用不同词汇，编辑右列。
