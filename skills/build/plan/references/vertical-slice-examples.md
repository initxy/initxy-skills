# 垂直切片示例

计划快变成水平拆分时读取本文件。

## 好的垂直切片

| 功能 | 切片 |
|---|---|
| 用户注册 | 表单 + API + 校验 + 持久化 + 确认反馈 |
| CSV 导入 | 上传 + 解析主路径 + 保存 + 展示结果 |
| 评论回复 | 界面操作 + 接口 + 存储 + 渲染嵌套回复 |
| 计费告警 | 事件输入 + 规则判断 + 通知输出 |

每个切片都应该能单独演示。

## 坏的水平切片

- 「先建所有数据库表」
- 「先写完所有后端接口」
- 「先做完所有前端页面」
- 「最后统一补测试」

## 步骤模板

```markdown
| # | 步骤 | 依赖 | 并行性 | 验证 |
|---|---|---|---|---|
| 1 | 为 <行为> 写失败测试 | - | 不可并行 | `<test command>` 因预期原因失败 |
| 2 | 实现最小 <行为> | 1 | 不可并行 | `<test command>` 通过 |
```
